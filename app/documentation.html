Overview

Your code must be in the following syntax in order to work:

function yourScript() {
    return [0, 1];  // x, y coordinates, with 0, 0 being the top left corner
}

Keep all code in a single function (inner functions are allowed).

If your code returns an invalid move (like a coordinate already hit, or one outside the board) it will count against your total moves but otherwise be ignored.

You have access to the self variable to save state between moves if you wish, as well as access to game state.

Logging is available through console.log(message). You can also use console.error(message), console.warn(message), and console.debug(message).

All built-in game state variables are backed up prior, and restored after, your function runs. This prevents tampering.

Functions

Function Description
self.getBoardWidth() An integer representing the width of the board
self.getBoardHeight() An integer representing the height of the board
self.getMoves() An array representing all of your previous moves, in order. Ex: [[0,0], [5,2], ...]
self.getBoard() A 2d array describing the known board state. Each element in the array has properties of: isHidden, isHit, isMiss, isSunk.
self.getShips() An array describing the known ship states. Each element in the array has properties of: size, isSunk.
